## 2026-06-09 — Session 1: Initial Creation

**Source material:** `product-brief.docx` + `game-modules.docx` (extracted and used as primary inputs)

**Intent:** Create — Fast path (existing DOCX used as source, full draft with [ASSUMPTION] tags)

**Key decisions inherited from source documents:**
- Game name: Bomb Squad (working title, inspired by Keep Talking and Nobody Explodes)
- Platform: Web-only (browser), self-hosted, no mobile app in v1
- Primary use case: Private, invite-only internal team building
- Primary game mode: Relay Race (competitive, cumulative time)
- Voice: LiveKit WebRTC SFU (decided to keep after party mode review)
- Tech stack locked: React 18+, Three.js/R3F, Zustand, Node.js/Fastify, Socket.IO, Redis, PostgreSQL
- V1 modules: Wires, Button, Keypads, Simon Says, Memory, Morse Code
- Out of scope (v1): mobile-native, persistent accounts, public matchmaking, parallel defuse, Hot Seat Rotation mode
